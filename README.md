# Airay_app_by_NeurovaSpace
🗓️ Airay
A minimalist, aesthetic calendar application built for the desktop with  Electron, HTML, and CSS.

Airey is a distraction-free scheduling tool designed with a focus on cinematic UI and simplicity. Built using Electron, it brings the flexibility of web technologies to a native desktop experience.

✨ Features
Minimalist Design: A clean, "less-is-more" interface to help you focus on your schedule.

Cinematic Aesthetic: Carefully curated typography and layout for a premium visual feel.

Desktop Native: Powered by Electron for a seamless experience on Windows, macOS, or Linux.

Lightweight: Built with standard web technologies (HTML5, CSS3, and JavaScript).

🚀 Getting Started
Prerequisites
You will need Node.js installed on your computer to run this project.
